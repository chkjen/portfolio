About

Thanks for stopping by! I'm Chuck and I'm a Web Developer.

Since graduating college with an Associates in Applied Science: Program Engineering, I've been working on many fun and exciting projects. This has allowed me to learn many different technologies to create websites that are both engaging and captive to users. I find myself to be both highly adaptive as well as quick to learn new technologies.

Recently, in an effort to expand my knowledge, I completed a full stack web development bootcamp on Udemy (December 2019). This bootcamp included technologies from the following areas: HTML, CSS, Bootstrap, Javascript ES6, DOM maniulation, JQuery, Unix Command Line, Node.js, Express, Git/GitHub version control, API creation/connection and RESTful methodology, EJS, Databases (SQL & MongoDB), Mongoose, Authentication and Security as well as React.js.

I thouroughly enjoy complex projects that test and expand my coding and development skills. The satisfying feeling that occurs when a breakthrough is made on a problem is awesome!

One of my most exciting goals for this year is to colaborate more on open source projects to give back to the community for all the learning they've provided. I also have several side projects that I'm working that will help grow my knowledge of SEO and online sales as well as API creation.

I built this site from scratch utilizing HTML, CSS, Javascript and Node.js via VSCode. Daily backups were made to a local Git repository and then backed up to the github cloud. The site itself is hosted on Heroku utilizing a custom domain configuration. The API keys are stored in an enironment variable file which has been added to the .gitignore file to prevent unwanted distribution. All of the dependancies can be found in the package.json file.
