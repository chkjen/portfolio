About

Thanks for stopping by! I'm Chuck and I'm a web developer

Since graduating college in 2006 with an Associates in Applied Science: Program Engineering, I've been working on many fun and exciting projects. I find myself to be both highly adaptive as well as quick to learn new technologies.

Recently, in an effort to expand my knowledge, I completed a full stack web development bootcamp on Udemy (December 2019). This bootcamp included technologies from the following areas: HTML, CSS, Bootstrap, Javascript ES6, DOM maniulation, JQuery, Unix Command Line, Node.js, Express, Git/GitHub version control, API creation/connection and RESTful methodology, EJS, Databases (SQL & MongoDB), Mongoose, Authentication and Security and React.js.

I thouroughly enjoy complex projects that test and expand my coding and development skills. While I enjoy working with a group of people on projects the most, I'm fully capable of working autonmously.

One of my most exciting goals for this year is to colaborate more on open source projects to give back to the community for all the learning they've provided. I also have several side projects that I'm working that will help grow my knowledge of SEO and online sales as well as API creation.
